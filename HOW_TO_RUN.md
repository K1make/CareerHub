# Как запустить проект CareerAI

Для запуска данного проекта на вашем локальном компьютере, выполните следующие шаги:

1. **Установите зависимости** (если вы только что скачали проект). Откройте терминал в папке проекта и выполните команду:
   ```bash
   npm install
   ```

2. **Запустите локальный сервер для разработки.** В этой же папке выполните команду:
   ```bash
   npm run dev
   ```

3. **Откройте приложение в браузере.** После успешного запуска в терминале появится ссылка (обычно `http://localhost:5173/`). Перейдите по ней или зажмите `Ctrl` и кликните по ссылке в терминале.
