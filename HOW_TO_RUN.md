# Как запустить проект CareerHub

Для полноценной работы проекта вам нужно запустить **две** части: серверную (Backend на Django) и клиентскую (Frontend на React/Vite). Для этого потребуется открыть два разных терминала.

### Шаг 1. Запуск Backend (Django)

1. Откройте первый терминал в папке проекта.
2. Активируйте виртуальное окружение (если оно не активировано):
   ```bash
   # Для Windows:
   venv\Scripts\activate
   ```
3. Запустите сервер:
   ```bash
   python manage.py runserver
   ```
   *Сервер будет доступен по адресу: http://127.0.0.1:8000/*

### Шаг 2. Запуск Frontend (React/Vite)

1. Откройте **второй** терминал в этой же папке проекта.
2. Установите зависимости (если вы запускаете проект впервые):
   ```bash
   npm install
   ```
3. Запустите локальный сервер для разработки:
   ```bash
   npm run dev
   ```
4. **Откройте приложение в браузере.** После успешного запуска в терминале появится ссылка (обычно `http://localhost:5173/`). Перейдите по ней или зажмите `Ctrl` и кликните по ссылке в терминале.
